# Running Tests

Maven:

* navigate to project folder in terminal
* run : mvn clean test

TestNG

  * right click on testng.xml
  * run testng.xml
  
 JUnit Jupiter
 
 (Does Not Support Parallel Execution)
 
  * open MyFirstJUnitTestCase.java
  * run class as JUnit test
  * run method as JUnit test
